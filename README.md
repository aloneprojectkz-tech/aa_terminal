# 📬 TempMail Telegram Bot

Бот создаёт временные почтовые ящики через **mail.tm API** и пересылает все входящие письма пользователю в Telegram.

---

## Структура проекта

```
mailbot/
├── bot.py          # Точка входа
├── config.py       # Настройки (env-переменные)
├── database.py     # Работа с PostgreSQL (asyncpg)
├── mailtm.py       # Клиент mail.tm API
├── handlers.py     # Все обработчики aiogram
├── keyboards.py    # Клавиатуры
├── scheduler.py    # Фоновый поллинг почты
├── requirements.txt
├── Dockerfile
└── .env.example
```

---

## Быстрый старт

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```

### 2. Создайте `.env`
```bash
cp .env.example .env
```
Заполните:
```
BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=ваш_telegram_id   # можно несколько через запятую
DATABASE_URL=postgresql://...
POLL_INTERVAL=30            # интервал проверки почты (сек)
```

### 3. Запустите
```bash
python bot.py
```

### Docker
```bash
docker build -t tempmail-bot .
docker run --env-file .env tempmail-bot
```

## Функции

### 👤 Пользователь
| Кнопка | Действие |
|--------|----------|
| 📧 Создать почту | Создаёт новый временный ящик (лимит 5 шт.) |
| 📬 Мои почты | Список активных ящиков |
| ℹ️ Помощь | Инструкция |

- Нажав на ящик в списке — можно скопировать адрес/пароль или удалить.
- Все входящие письма приходят автоматически каждые ~30 секунд.

### 👑 Администратор
Добавьте свой Telegram ID в `ADMIN_IDS`.

| Кнопка | Действие |
|--------|----------|
| 📊 Статистика | Кол-во пользователей, ящиков, писем |
| 👥 Пользователи | Список с пагинацией, бан/разбан |
| 📩 Активные почты | Все активные ящики с owner ID |
| 📢 Рассылка | Отправить HTML-сообщение всем пользователям |

---

## База данных

Таблицы создаются автоматически при первом запуске:

- `users` — зарегистрированные пользователи
- `mailboxes` — почтовые ящики
- `messages` — доставленные письма (дедупликация)
